Issue creation guidelines

  • Title string must exceed 5 characters.
  • Description must exceed 10 characters.
  • URLS, materials are optional.
  • If URLS or materials are provided they must be valid URL strings.
  • Materials should point towards an image file (png, jpeg, gif, tiff).
  • Dataset-ids list can't be empty.
  • Dataset-id must be in the form datasetid#version or datasetid.Vversion.
